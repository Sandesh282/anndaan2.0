Food Donation Backend API

Welcome to the Food Donation Backend API. This service provides endpoints for managing food donation requests with OTP verification.

Available Endpoints:

POST
/api/submit-application

Submit a food donation application and generate OTP

POST
/api/verify-otp

Verify the OTP sent to the phone number

POST
/api/save-donation

Save the donation request after OTP verification

How to Use:

  1. Submit application with user details
  2. Check console for generated OTP
  3. Verify OTP using the verification endpoint
  4. Save the donation request

Note: OTPs are logged to the server console for demonstration purposes.
